## User Story

Como **Anderson Guimarães**, quero **`https://www.rfgcorretora.com.br` ativo** com SSL Let's Encrypt automático, GA4 + Meta Pixel reais funcionando, Lighthouse mobile ≥95 em produção, **8 CTAs WhatsApp testados em meu celular** com mensagens canônicas corretas, redirect raiz→www e headers de segurança nota mínima A — para que **a landing v1 esteja oficialmente no ar capturando leads via WhatsApp** e Quality Gates §9 do PRD passem todos.

Story de **go-live**. Executor exclusivo é **@devops (Gage)** por Agent Authority (push, MCP, deploy, DNS são exclusividade DevOps).
